O que é o Sistema de Filmes?

O sistema de gerenciamento de filmes permite a usuários cadastrar títulos, realizar locações e acompanhar quais filmes já foram assistidos. Ideal para quem quer manter uma videoteca pessoal ou controlar o que já viu.

Funcionalidades Principais

  • Cadastro de Filmes: Insira novos títulos com informações detalhadas.
  • Locação de Filmes: Marque os filmes que deseja assistir ou já assistiu.
  • Controle de Status: Veja quais filmes estão pendentes ou já assistidos.
  • Gerenciamento por Usuário: Cada usuário pode visualizar sua própria lista de filmes.

Vantagens de Usar o Sistema

  • Organização: Mantenha todos os seus filmes cadastrados em um só lugar.
  • Histórico de Visualização: Saiba quais filmes já foram assistidos.
  • Facilidade: Interface simples e intuitiva para cadastro e controle.
  • Produtividade: Ideal para quem quer aproveitar melhor seu tempo livre assistindo bons filmes.
